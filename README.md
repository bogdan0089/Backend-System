# Backend-System

## Description
Backend system for managing an online shop, built with Python and FastAPI.  
Supports clients, products, orders, and transactions, demonstrating CRUD operations, database migrations with Alembic, and clean backend code organization.

## Technologies
- Python 3.11
- FastAPI
- SQLAlchemy
- Alembic (database migrations)
- PostgreSQL
- Docker & Docker Compose
